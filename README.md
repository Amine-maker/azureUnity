# Azure unity 

## Prérequis
 - Azure CLI https://docs.microsoft.com/fr-fr/cli/azure/install-azure-cli-windows?tabs=azure-cli
 - npm 
 - nodejs


 ## Lancer le projet
```
cd ./back/backAzurePlatform
npm i 
cd ./react-platform
npm i 

Puis
npm start
à la racine du projet, ce qui démarrera les deux serveurs
```
